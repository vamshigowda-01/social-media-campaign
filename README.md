# Social Media Campaign – Fitness Week

## Objective
The objective of this project is to design a week-long social media campaign using carousel and motion posts to promote fitness awareness and encourage people to stay active.

## Tools Used
- Canva
- Adobe Illustrator (optional)
- Canva Animation tools

## Project Description
This project presents a 7-day fitness social media campaign. Each slide represents a different day of the fitness challenge and includes workout tips and motivational messages.

The campaign uses a carousel format where users can swipe through slides to follow the weekly challenge. Motion effects such as fade, rise, and zoom were applied to create animated posts.

Brand consistency was maintained by using the same colors, fonts, and design style across all slides. Interactive elements like call-to-action messages were added to encourage audience engagement.

## Outcome
The final campaign includes:
- Carousel slides for social media
- Animated motion posts
- A consistent fitness-themed design suitable for platforms like Instagram and Facebook.
